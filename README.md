# Supporting
- Android 5.1-11
- SuperSU

# Notes
- Now Android S not supported fully.

# Instructions
- Download and install app
- Grant all permissions (including root)
- Navigate to Menu>Chroot Manager>Install>Download official image..>Wait
- Reboot your device
- Profit!
