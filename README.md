# Supporting
- Android 7-12
- SuperSU

# Instructions
- Download and install app
- Grant all permissions (including root)
- Navigate to Menu > Chroot Manager > Install > Download > Enter a direct link to choot traball > Wait
- Reboot your device
- Profit!
